# Code of Conduct

This project welcomes contributions and discussion from everyone. To keep it a positive space:

- Be respectful and constructive in issues, pull requests, and discussions.
- No harassment, personal attacks, or discriminatory language.
- Stay on topic — this repo is about the installer and its supported models/tools.

Reports of unacceptable behavior can be sent privately through the maintainer's GitHub profile. Violations may result in comments being removed or contributors being blocked from the repository.
